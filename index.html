GT Sport Randomiser

For creating random races for "fun" GT Sport lobbies. Either go fully random with the Randomise button or use the Next Track button to make sure you don't visit a track twice.

Generate a fully random list of tracks, car groups, and other settings. Needs to be used first to setup the random list.

Go to the next track in the random list, with the Car Group and other settings remaining randomised.

Class

Laps
























































N.B. There are no N900 cars available without tuning, therefore this is turned off by default.
